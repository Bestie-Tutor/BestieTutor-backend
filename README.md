#### How to Install

<pre>
git clone https://github.com/Bestie-Tutor/BestieTutor-backend.git
cd backend
cd BestieTutor-backend
npm install -g nodemon
yarn add express
yarn add dotenv
yarn add axios
yarn add cors
nodemon index.js
</pre>
